# DidaWorks

Free and open-source tools designed around real teaching workflows.

## Current project

### [DidaRec](https://github.com/DidaWorks/didarec)

A free and open-source screen recorder for educators.

## DidaCommons

[DidaCommons](https://github.com/DidaWorks/commons) is the community and shared-resource space surrounding DidaWorks.

## Principles

- Free and open-source
- Designed with educators in mind
- Privacy-conscious
- Accessible
- Uncomplicated
- Useful without subscriptions or mandatory accounts

## Project status

DidaWorks is currently an independent portfolio and open-source project under active development.
